# ![Django CRUD App Group Project](./assets/hero.png)

## About

***Instructors should see the Instructor Guide for more configuration details.***

In this project, students will work in a group to build an authenticated Django application with full CRUD on a PostgreSQL database.

## Prerequisites

This project requires an understanding of full-stack authenticated applications built using Django and PostgreSQL.
